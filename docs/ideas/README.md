# websites — idea backlog & lifecycle

> **Status:** `ideas`
>
> Generated by substrate-kit. Capture ideas here so they live in the repo, not in
> chat. Nothing here is approved until it graduates. A **conveyor, not a graveyard**:
> every idea ends implemented, on a roadmap, in discussion, or explicitly rejected.

## Lifecycle

```
(1) INTAKE   capture the idea (raw -> captured)
(2) MAP      name the owning area, rough size, rough risk
(3) ROUTE    -> quick-win | structured plan | discuss-first (question router)
(4) GROOM    pull one routable idea forward each session
(5) OUTCOME  implemented | on a roadmap | in discussion | rejected
```

## What this repo is

The **websites** repo for the menno420 estate — one repo, multiple Railway
services (share code, not a process):
- **control-plane** (`app/`) — private readiness board + journal browser over
  `superbot`, `superbot-next`, `substrate-kit`, and `websites` itself, live from
  the GitHub API. Full doc: `docs/site.md`.
- **botsite** (`botsite/`) — the public marketing + reference site. Doc:
  `docs/botsite.md`.
- **dashboard** (`dashboard/`) — the private developer dashboard: read-only
  oversight of the bot's own inventory (functions/commands/settings/access/env/
  ideas/bugs), with the live-bot-write control panel stubbed by design. Doc:
  `docs/dashboard.md`.

Deployment (all three services): `docs/deployment.md` + each service's doc.

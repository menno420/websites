what this page is

The docs/ideas/ conveyor across the fleet. Titles and one-line summaries are parsed from each idea file; every idea deep-links to its source on GitHub (and to the in-app markdown view). Repos without an ideas directory are simply absent below. State counts cover each repo's newest enriched files (honest scope); unstated = no lifecycle front-matter. With no state filter, each repo's built/retired ideas sit collapsed under its live captured/planned/unstated list.
